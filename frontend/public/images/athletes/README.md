# 📸 Imágenes de Atletas

Esta carpeta es para guardar las imágenes de los atletas.

## 📝 Cómo usar:

1. **Guarda las imágenes aquí** con nombres descriptivos:
   - `messi.jpg`
   - `ronaldo.png`
   - `neymar.webp`

2. **En el Admin**, usa la URL relativa:
   - `/images/athletes/messi.jpg`
   - `/images/athletes/ronaldo.png`

3. **Formatos soportados:**
   - `.jpg` (Recomendado)
   - `.png`
   - `.webp`
   - `.gif`

## 📐 Tamaño recomendado:
- **Ancho:** 400-500px
- **Alto:** 500-600px
- **Peso:** < 200KB

## ⚠️ Importante:
- Usa nombres sin espacios ni caracteres especiales
- Mantén las imágenes optimizadas para web
- Si no hay imagen, se mostrará el placeholder automáticamente
