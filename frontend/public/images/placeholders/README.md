# 🎭 Placeholders para Cartas

Estos son los placeholders que se muestran cuando un atleta no tiene imagen.

## 📁 Archivos:
- `common-placeholder.png` - Para cartas comunes
- `rare-placeholder.png` - Para cartas raras  
- `epic-placeholder.png` - Para cartas épicas
- `legendary-placeholder.png` - Para cartas legendarias

## 🎨 Diseño:
- Fondo según la rareza
- Icono de usuario genérico
- Tamaño: 400x500px
- Formato: PNG con transparencia

## 🔄 Uso automático:
El sistema mostrará estos placeholders cuando:
- No hay `image_url` en la base de datos
- La URL de la imagen no carga
- Hay un error al cargar la imagen

## 📝 Nota:
Puedes reemplazar estos archivos con diseños personalizados
manteniendo los mismos nombres.
