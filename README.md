# Orcabet - Fantasy Orcasitas

Aplicación web de apuestas fantasy para el fútbol local Orcasitas, construida con arquitectura 100% serverless.

## Stack Tecnológico
- **Frontend**: React 18 + TailwindCSS + Firebase
- **Backend**: Firebase Auth + Firestore
- **Deploy**: GitHub Pages

## Estructura
- `/frontend` - Aplicación React
- `.github/workflows` - CI/CD para deploy automático

## Deploy
El deploy se realiza automáticamente en cada push a main a través de GitHub Actions.
